Q1.Define the terms: Website, Webpage, Web browser, Web server, HTML, CSS

Website: A website is a collection of interconnected webpages that are typically accessed through a uniform resource locator (URL)
and hosted on a web server. It is a digital platform containing various types of content, such as text, images, multimedia elements,
and interactive features, designed for users to access and navigate using a web browser.

Webpage: A webpage is a single document or file within a website. It is a specific location on the web that displays information or content.
Webpages are written using languages like HTML (Hypertext Markup Language) and often incorporate other technologies such as CSS
(Cascading Style Sheets) and JavaScript to enhance their appearance and functionality.

Web browser:A web browser is a software application that allows users to access, view, and interact with information on the internet.
It retrieves webpages from web servers and renders them on a user's device, such as a computer or mobile device. Popular web browsers
include Google Chrome, Mozilla Firefox, Microsoft Edge, and Safari.

Web server: A web server is a computer or software application that stores and delivers webpages and other files to web browsers upon request.
It hosts the website's files and responds to requests made by users' web browsers. Web servers use protocols like
HTTP (Hypertext Transfer Protocol) to facilitate the communication between the server and the browser.

HTML (Hypertext Markup Language): HTML is the standard markup language used for creating the structure and content of webpages.
It provides a set of tags or elements that define the various elements of a webpage, such as headings, paragraphs, links, images, tables,
forms, and more. HTML tags are interpreted by web browsers to display the webpage content in a structured manner.

CSS (Cascading Style Sheets):CSS is a style sheet language used for describing the presentation and visual appearance of HTML documents.
It allows web developers to control the layout, colors, fonts, and other visual aspects of webpages. By separating the content (HTML)
from the presentation (CSS), it enables consistent styling across multiple webpages of a website and enhances the overall user experience.
